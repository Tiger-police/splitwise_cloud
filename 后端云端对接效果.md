### 4.3 触发调度
#### 登录拿token:
`nss-d` :
```bash
TOKEN=$(curl -s -X POST http://127.0.0.1:8010/api/v1/login \
  -H "Content-Type: application/json" \
  -d '{"username":"admin","password":"admin123"}' | python -c "import sys,json; print(json.load(sys.stdin)['access_token'])")
echo $TOKEN

```
### 触发调度
```yaml
TRIGGER_RESPONSE=$(curl -s -X POST http://127.0.0.1:8010/api/v1/schedule/trigger \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $TOKEN" \
  -d '{"model_type":"llama-3.2-3b"}')
echo $TRIGGER_RESPONSE
```

### 取`task_id` 
```yaml
TASK_ID=$(echo "$TRIGGER_RESPONSE" | python -c "import sys,json; print(json.load(sys.stdin)['task_id'])")
echo $TASK_ID
```

### 查状态
```yaml
curl -H "Authorization: Bearer $TOKEN" \
  http://127.0.0.1:8010/api/v1/schedule/tasks/$TASK_ID
```
